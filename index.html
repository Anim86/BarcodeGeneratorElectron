Guida Rapida

Scopri come utilizzare al meglio EAN Demon Generator

1. Inserimento Codici

Nel pannello di sinistra, puoi inserire i codici EAN manualmente scrivendoli uno per riga. L'applicazione convaliderà all'istante la validità sintattica di ogni codice e calcolerà la cifra di controllo corretta se mancante.

2. Configurazione e Anteprima

Al centro viene mostrata l'anteprima in tempo reale. Puoi regolare le dimensioni del barcode, i margini di abbondanza, il colore (RGB, CMYK, HSL o nero K100%) e le opzioni del testo HRI. L'app ti avvisa in caso di contrasto insufficiente o margini ridotti.

3. Importazione ed Elaborazione Batch

A destra, puoi caricare file Excel (XLSX) o CSV. Se il file contiene una colonna con i nomi dei prodotti ed una con i codici EAN, l'app nominerà automaticamente i file generati (es. "NomeProdotto_EAN.png"). Puoi avviare l'esportazione massiva in un clic!
