# EAN Pro — Professional Barcode Generator

EAN Pro è un'applicazione desktop professionale per la generazione di codici a barre EAN-13 ed EAN-8 in alta risoluzione, ottimizzata per il mondo della prestampa e del packaging.

Puoi scaricare l'ultima versione stabile per Windows e macOS dalla sezione **Releases**:

👉 **[Scarica EAN Pro (Ultima Versione)](https://github.com/Anim86/BarcodeGeneratorElectron/releases/latest)**

*(Nota: Se il repository è privato, devi aver effettuato l'accesso a GitHub per scaricare i file).*

---

## 🚀 Caratteristiche Principali
- **Esportazione Professionale**: Supporto per PDF/X-1a, EPS (vettoriale), TIFF e JPEG in CMYK.
- **Gestione Colore**: Possibilità di forzare il Nero K100 (Grigio) per una stampa perfetta.
- **Batch Processing**: Genera centinaia di barcode contemporaneamente importando un file Excel (.xlsx) o CSV.
- **HRI Personalizzabile**: Controllo totale sul testo leggibile (Font OCR-B incluso, posizionamento e formattazione GS1).
- **Anteprima Real-Time**: Verifica istantaneamente la validità del codice e il contrasto colore.

## 🛠️ Requisiti di Sistema
- **Windows**: Windows 10 o superiore.
- **macOS**: macOS 10.12 o superiore (Intel o Apple Silicon).
  - *Nota*: Per le esportazioni PDF/EPS su Mac, è consigliato installare Ghostscript (`brew install ghostscript`).

## 👨‍💻 Sviluppo
Se desideri compilare l'app dai sorgenti:
1. Clona il repository.
2. Esegui `npm install`.
3. Per avviare: `npm start`.
4. Per compilare: `npm run build:win` o `npm run build:mac`.
